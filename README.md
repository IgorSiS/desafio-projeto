# Cadastro de Projeto

### Tecnologias Utilizadas

* Java 11
* Spring Boot
* Postgres
* Padrão de Projeto Clean Architecture
* MapStruct
* Docker
* Groovy
* Spock - Para testes unitários
* Logback
* Heroku
* Github


* OBS1: Contém 3 casos de teste apenas para demonstraçao
